    JSON importar / exportar

    «Export JSON» gera um ficheiro reimportável a partir dos eventos finding na timeline (o buffer do mapa mantém os últimos 200 eventos). Compatível com export JSON da UI (findings[] com priority, mitre[].id). Máx. 600 achados por importação.

    Legenda
    • Quadrado laranja — técnica MITRE (Txxxx) ou número do achado; clique para detalhe.
    • ↑ Ramo acima da linha / ↓ abaixo — alterna a cada achado.
    • HIGH · MED · LOW — borda esquerda do cartão.

    Pipeline horizontal por fases. Cada achado gera um ramo ↑ cima / ↓ baixo em alternância. O quadrado laranja mostra a técnica MITRE (T1595) ou nº do achado — clique para ver os links ATT&CK e OWASP. À esquerda, o separador JSON abre importar/exportar (interruptor estilo fila + campo + Export JSON). À direita, Legenda abre o painel de cores e símbolos. O mini-pipeline (sticky) mostra Kali ON / Kali OFF conforme «Modo Kali» na corrida. Usa os filtros na matriz.
